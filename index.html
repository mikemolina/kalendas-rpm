    Building from the SRPM

    Compilando desde el SRPM

    Compilação a partir do SRPM

    Need to install basic development tools

    Necesita instalar los paquetes básicos de compilación

    Instale os pacotes necessários para compilação

    $ sudo yum install @development-tools
$ sudo yum install fedora-packager yum-utils

    create the directory tree building

    crear los directorios compilación

    criar o pasta de compilação

    $ rpmdev-setuptree

    download and install the SRPM package in ~/rpmbuild

    descargar e instalar el paquete SRPM en ~/rpmbuild

    baixar e instalar o pacote SRPM em ~/rpmbuild

    $ yumdownloader --source kalendas
$ rpm -ivh kalendas-*.src.rpm

    Next, build the RPM package

    En seguida, compilar el paquete RPM

    Em seguida, compilar o pacote RPM

    $ rpmbuild -bb ~/rpmbuild/SPECS/kalendas.spec

    the RPM are then available in ~/rpmbuild/RPMS/noarch. Now you can review and install

    el RPM estara disponible en ~/rpmbuild/RPMS/noarch. Ahora puede revisarlo e instalarlo

    o pacote RPM estará disponível em ~/rpmbuild/RPMS/noarch. Instale e teste o seu pacote

    $ rpmls ~/rpmbuild/RPMS/noarch/kalendas-*.noarch.rpm
$ sudo rpm -ivp ~/rpmbuild/RPMS/noarch/kalendas-*.noarch.rpm

    More information on RPM building in How to create an RPM package.

    Más información sobre la compilación de un RPM en How to create an RPM package.

    Mais informações sobre a compilação de pacotes RPM em How to create an RPM package.
