    What is?

    Qué es?

    O que é?

    kalendas is a perl script to make calendar calculations. The computations are developed on three systems of dating: the Julian calendar, the Gregorian calendar and the system of numbering of Julian day/date.

    kalendas es un perl script para realizar cálculos de calendario. Los cómputos están desarrollados sobre tres sistemas de datación: el calendario Juliano, el calendario Gregoriano y el sistema de numeración de día/fecha Juliana.

    kalendas é um perl script para realizar cálculos de calendário. Os cómputos estão desenvolvidos sobre três sistemas de datação: o calendário Juliano, o calendário Gregoriano e o sistema de numeração de dia/data Juliana.
